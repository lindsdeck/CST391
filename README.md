## Introduction

Throughout CST-391, I learned how to build full-stack web applications using modern JavaScript frameworks and REST APIs. The course started with foundational web development activities and progressed into backend development, database integration, Angular, React, routing, API communication, and full CRUD functionality. My main course project was the Little Care Finder application, which allows users to manage childcare center information through a connected frontend and backend system.

## Presentation Artifacts

This repository contains the code, documentation, screenshots, and milestone reports completed throughout the course. The major project artifacts include:

- Backend REST API using Express and Node.js

- Screenshots and milestone reports  
- Screencast demonstrations

The final Little Care Finder application demonstrates:

- Listing childcare centers  
- Viewing details for a single center  
- Creating a new center  
- Updating an existing center  
- Deleting a center

- API communication using Axios  
- Persistent data storage using MySQL  

## Challenges Encountered

Some challenges included resolving Git synchronization issues, debugging API request errors such as incorrect endpoint formatting, and handling route parameters in React Router. I also encountered issues with asynchronous data loading, particularly ensuring that data was properly retrieved before rendering components.

## Lessons Learned

This course helped me better understand how frontend and backend applications work together in a full-stack environment. I expanded my knowledge of how REST APIs communicate with databases and how React and Angular manage components and state. I also gained experience with routing, dynamic rendering, and handling user input through controlled components. In addition, I became more confident using tools such as Visual Studio Code, GitHub, MySQL Workbench, Postman, and the terminal.
